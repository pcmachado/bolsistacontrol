# Manual do Bolsista

## Objetivo do Perfil

Registrar atividades, submeter frequencias mensais e acompanhar pagamentos.

## Acessos Principais

1. `attendance.my` - Registros de frequencia
2. `attendance.submissions.my` - Submissoes mensais
3. `attendance.reports.index` - Relatorio mensal
4. `attendance.reports.final.create` - Relatorio final
5. `payments.my` - Meus pagamentos

## Rotina Recomendada

1. Registrar frequencia diariamente
2. Revisar horas e descricoes semanalmente
3. Submeter o mes dentro do prazo
4. Acompanhar status da submissao
5. Conferir pagamentos e confirmar recebimento quando aplicavel

## Passo a Passo

## 1) Registrar frequencia

1. Acesse `attendance.my` ou `attendance.create`
2. Informe data, horario, descricao e carga horaria
3. Salve o registro

## 2) Submeter frequencia mensal

1. Acesse `attendance.submissions.my`
2. Selecione o periodo
3. Revise os registros vinculados
4. Execute envio da submissao

## 3) Emitir relatorio mensal/final

1. Acesse `attendance.reports.index`
2. Selecione submissao e formato de saida
3. Para relatorio final, use `attendance.reports.final.create`

## Erros Comuns

1. Mes sem registros suficientes para submissao
2. Dados inconsistentes de horario/carga horaria
3. Tentativa de edicao apos status de envio/homologacao

## Boas Praticas

1. Nao deixar lancamentos para o fim do mes
2. Escrever descricoes objetivas e auditaveis
3. Guardar comprovantes relacionados as atividades
