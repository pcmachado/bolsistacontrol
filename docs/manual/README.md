1. Registro de frequencia pelo bolsista
2. Submissao mensal
3. Homologacao pela coordenacao
4. Consolidacao financeira e pagamentos
5. Gestao academica (projetos, cursos, disciplinas e turmas)

## Regras Gerais de Uso

1. Sempre validar filtros antes de exportar dados
2. Usar "Limpar" para remover filtros ativos
3. Conferir o perfil visivel no topo antes de operacoes sensiveis
4. Evitar manter abas antigas abertas durante alteracoes em lote
