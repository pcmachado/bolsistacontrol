# ProBolsas – Sistema de Gestão de Bolsas, Frequência e Pagamentos Acadêmicos

## Titular

Pessoa Física: Paulo César Machado

## Tipo de Programa

Sistema de gestão administrativa e educacional com controle financeiro.

## Linguagens Utilizadas

- PHP (Laravel)
- JavaScript
- SQL (PostgreSQL)

## Finalidade

O sistema tem como objetivo gerenciar:

- Frequência de bolsistas
- Submissões mensais
- Homologação institucional
- Gestão de turmas e alunos
- Controle financeiro de bolsas e auxílios

## Características Técnicas

- Arquitetura MVC
- Backend Laravel
- Banco de dados relacional
- Interface web responsiva

## Diferenciais

- Controle sequencial mensal (não permite inconsistência de períodos)
- Integração acadêmico-financeira
- Controle por perfis (admin, coordenação, bolsista)
- Regras automatizadas de validação

## Área de Aplicação

- Educação
- Administração pública
- Gestão de projetos acadêmicos
