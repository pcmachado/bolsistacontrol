# Resumo Funcional do Sistema

O sistema ProBolsas é composto por três módulos principais:

## 1. Frequência de Bolsistas

- Registro diário
- Submissão mensal
- Homologação por coordenação

## 2. Gestão de Turmas e Alunos

- Cadastro de alunos
- Lançamento mensal de frequência
- Submissão por turma

## 3. Financeiro

- Geração automática de pagamentos
- Controle de status (pending, sent, overdue, paid)
- Relatórios e dashboards

## Regras de Negócio

- Submissão mensal sequencial obrigatória
- Rejeição retorna pacote completo
- Pagamento depende de submissão válida
